Condiciones para ingreso de los valores

Condiciones para el valor A

  • El valor debe ser mayor a 0 (A>0)
  • El valor debe ser entero impar
  • El valor debe no debe ser divisible entre 3 o 5
  • Se recomienda usar valores con fórmula A=10x+1

Condiciones para el valor C

  • El valor debe ser mayor a 0 (C>0)
  • El valor debe cumplir con la codición C => C mod 8 = 5

Condiciones para el valor Xn

  • El valor debe ser mayor a 0 (Xn>0)

Condiciones para el valor M

  • El valor debe ser mayor a 0 (M>0)
  • El valor debe ser mayor a los demás valores ingresados(A,C,Xn)
